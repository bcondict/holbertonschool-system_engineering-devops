# 0x02. Shell, I/O Redirections and filters

### What do the commands head, tail, find, wc, sort, uniq, grep, tr do

- Head= 10 first lines in the command that you whant * head [OPTION] [file] *
- tail= 10 last lines in the command that you want * tail [OPTION] [file] *
- find= used to search and locate the list of file and directories *
- sort= is used to sort a file * sort [OPTION] [file] *

### How to redirect standard output to a file

- > redirects the output of a command to a file, replacing the existing contents of the file.

- >> redirects the output of a command to a file, appending the output to the existing contents of the file.
